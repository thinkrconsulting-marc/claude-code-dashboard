# Claude Code Knowledge Base Dashboard

## Overview
Dutch-language knowledge base dashboard for Claude Code documentation with multi-user auth, file uploads, AI categorization, GitHub repo management, and security scanning.

## Tech Stack
- Next.js 14 (App Router), TypeScript, Tailwind CSS, Prisma (PostgreSQL)
- Auth: NextAuth with email/password
- AI: Abacus AI LLM API (gemini-2.5-flash)
- Cloud storage for file uploads

## Design
- Dark mode forced (next-themes, enableSystem=false)
- Orange primary color (hsl 24 95% 53%)
- Font: Inter (display), JetBrains Mono (code)
- All UI text in Dutch (Nederlands)

## Key Pages
- `/dashboard` - Main overview with stats
- `/dashboard/zoeken` - Full-text search
- `/dashboard/templates` - CLAUDE.md templates
- `/dashboard/bladwijzers` - Bookmarks
- `/dashboard/commando-referentie` - Command reference
- `/dashboard/hoofdstuk/[id]` - Chapter viewer
- `/dashboard/repos` - GitHub repo library (filter by category)
- `/dashboard/repos/[id]` - Repo detail with README & security report
- `/dashboard/security` - Security scan dashboard
- Admin pages: upload, gidsen, gebruikers, tekst-invoer, repos

## Database Models
- User, Guide, Chapter, Section, ContentBlock, Bookmark, Upload, Template, GitHubRepo

## Important Patterns
- `chapter-content` CSS class on CardContent for chapter text visibility
- Dark mode contrast fixes in globals.css with !important overrides
- Hydration: suppressHydrationWarning on html/body, mounted state in forms
- GitHubRepo has enums: RepoCategory (SKILL, MCP_SERVER, TOOL, PLUGIN, OTHER), SecurityStatus (NOT_SCANNED, SAFE, WARNING, DANGEROUS)
- GitHub API rate limit: 60 req/hr without GITHUB_TOKEN env var
