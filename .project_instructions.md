# Claude Code Knowledge Base Dashboard

## Overview
Dutch-language knowledge base for Claude Code guide content. Multi-user auth with Admin/User roles.

## Tech Stack
- Next.js 14, React 18, TypeScript, Tailwind CSS, Prisma + PostgreSQL
- NextAuth v4 (credentials), Abacus AI LLM API (Gemini), S3 cloud storage

## Theme
- Dark mode default, orange/amber primary (`hsl(24, 95%, 53%)`)
- All UI text in Dutch (NL)

## Key Features
- 14-chapter guide viewer with code blocks, tables, bookmarks
- Global search, CLAUDE.md templates, command reference
- Admin: file upload with AI categorization, guide management, user management

## DB Schema
Models: User, Guide, Chapter, Section, ContentBlock, Template, TemplateSection, Bookmark, Upload, Command

## Routes
- `/login`, `/signup` — Auth pages
- `/dashboard` — Chapter grid
- `/dashboard/hoofdstuk/[id]` — Chapter viewer
- `/dashboard/zoeken` — Search
- `/dashboard/templates` — CLAUDE.md templates
- `/dashboard/bladwijzers` — Bookmarks
- `/dashboard/commando-referentie` — Command reference
- `/dashboard/admin/upload` — File upload + AI analysis
- `/dashboard/admin/gidsen` — Guide management
- `/dashboard/admin/gebruikers` — User management
